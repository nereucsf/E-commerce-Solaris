# Ecologic Solaris

## Energia Limpa para um Futuro Brilhante

Este projeto apresenta uma plataforma web inovadora focada em energia solar, desenvolvida para demonstrar a capacidade de criar soluções **Full Stack** com tecnologias modernas e eficientes. O Ecologic Solaris permite aos usuários controlar a produção de energia em tempo real, maximizar ganhos e contribuir para um futuro mais sustentável.

### Tecnologias Utilizadas

O desenvolvimento do Ecologic Solaris foi realizado utilizando as seguintes tecnologias:

*   **Frontend**: HTML, CSS, JavaScript
*   **Backend/Autenticação/Banco de Dados/Armazenamento**: Google Firebase
    *   **Firebase Authentication**: Gerenciamento de usuários e autenticação segura.
    *   **Cloud Firestore**: Banco de dados NoSQL para armazenamento e sincronização de dados em tempo real.
    *   **Cloud Storage for Firebase**: Armazenamento de arquivos e mídias.

### Funcionalidades Principais

*   **Autenticação de Usuários**: Sistema de login e registro seguro via Firebase Authentication.
*   **Controle de Produção de Energia**: Visualização e gerenciamento em tempo real da produção de energia solar.
*   **Economia e Sustentabilidade**: Informações e métricas sobre a redução da conta de luz e o impacto ambiental positivo.
*   **Interface Intuitiva**: Design responsivo e fácil de usar para uma experiência otimizada.
